FRENCH:

DEMARRAGE:

Vous pouvez directement démarrer le "run.bat" afin de le faire fonctionner,
Il est hébergé en localhost sur le port 8080
Le lien est donc: http://www.localhost:8080

DETAILS/CONFIGURATIONS:

Ce projet est un site web communiquant avec l'API de ce lien ci-dessous, dévellopé en Golang:
    -> https://www.groupietrackers.herokuapp.com

Toutes les explications/configurations associées à cela se situe dans le fichier nommé config.go
